## Proyecto – Ejercicio 10
## Descripción  
En este proyecto se resuelve el ejercicio 10, donde se calculan resultados a partir de los datos y se generan tablas con las posiciones finales de los participantes
### Instalación  
1. Crear un entorno virtual:
python -m venv venv

2. Activar el entorno virtual:
.\venv\Scripts\activate

3. Instalar las dependencias:
pip install -r requints.txt

### Ejecución  
1. Abrir Jupyter Notebook:
jupyter notebook

2. Ir a la carpeta notebooks  
3. Ejecutar el archivo .ipycorrespondiente

### Estructura  
- src/: contiene las funciones del programa  
- notebooks/: contiene los notebooks donde oks para poder utilizarlas.
